# Use Case Diagram - Orange Calculator App

## 2.1. Use Case Diagram

### 2.1.1. Actors (ผู้ใช้งานระบบ)
ระบบมีผู้ใช้งานหลัก 2 ประเภท:
1. **ผู้ใช้งาน (User)**: นักวิจัย เกษตรกร หรือผู้สนใจทั่วไปที่ต้องการคำนวณราคาส้มและดูข้อมูลราคา
2. **แอดมิน (Admin)**: ผู้ดูแลระบบที่มีสิทธิ์จัดการข้อมูลส้มและอัพเดทราคาจากแหล่งภายนอก

### 2.1.2. Use Cases

#### 2.1.2.1. ดูข้อมูลส้ม (View Orange Data)
การเรียกดูข้อมูลส้มแต่ละชนิดในระบบ โดยแสดงรายละเอียดเช่น ชื่อส้ม, ราคาต่อกิโลกรัม, ถิ่นกำเนิด, ฤดูกาล และคุณสมบัติต่างๆ ผู้ใช้สามารถเลือกดูข้อมูลส้มเพื่อประกอบการตัดสินใจก่อนทำการคำนวณราคา

#### 2.1.2.2. คำนวณราคาส้ม (Calculate Orange Price)
การคำนวณราคาส้มโดยผู้ใช้เลือกชนิดส้มและระบุน้ำหนัก (kg) ระบบจะคำนวณราคารวมอัตโนมัติจากราคาต่อกิโลกรัมในฐานข้อมูล และแสดงผลลัพธ์พร้อมรายละเอียด (ชนิดส้ม, น้ำหนัก, ราคา/kg, ราคารวม) ระบบจะบันทึกประวัติการคำนวณทุกครั้งลงฐานข้อมูล

#### 2.1.2.3. ดูราคาปัจจุบัน (View Live Prices)
การดึงและแสดงราคาส้มล่าสุดจากเว็บตลาดไทย (Talaadthai.com) แบบเรียลไทม์ ผู้ใช้สามารถดูราคาตลาดปัจจุบันของส้มแต่ละชนิด รวมถึงช่วงราคา (ราคาต่ำสุด-สูงสุด) และแหล่งที่มาของข้อมูล เพื่อเปรียบเทียบกับราคาที่คำนวณ

#### 2.1.2.4. ดูประวัติการคำนวณ (View Calculation History)
การเรียกดูประวัติการคำนวณราคาย้อนหลัง แสดงรายละเอียดแต่ละครั้ง เช่น วันที่และเวลา, ชนิดส้ม, น้ำหนัก, ราคาต่อ kg, และราคารวม ผู้ใช้สามารถดูสถิติการใช้งานและติดตามประวัติการคำนวณได้อย่างสะดวก

#### 2.1.2.5. ดูสถิติ (View Statistics)
การแสดงข้อมูลสถิติการใช้งานระบบ เช่น ส้มที่ได้รับความนิยมมากที่สุด, จำนวนครั้งที่คำนวณแต่ละชนิดส้ม, น้ำหนักรวมที่คำนวณ และมูลค่ารวม ข้อมูลแสดงในรูปแบบกราฟและตารางเพื่อให้เข้าใจง่าย

#### 2.1.2.6. จัดการข้อมูลส้ม (Manage Orange Data) - Admin
การจัดการข้อมูลส้มในระบบโดยแอดมิน สามารถเพิ่มส้มชนิดใหม่, แก้ไขข้อมูล (ชื่อ, ราคา, รายละเอียด), หรือลบส้มที่ไม่ใช้งานออกจากระบบ การเปลี่ยนแปลงจะมีผลทันทีกับการคำนวณและการแสดงผล

#### 2.1.2.7. Scrape ราคาจากเว็บ (Scrape Prices from Web) - Admin
การดึงข้อมูลราคาส้มจากเว็บตลาดไทยอัตโนมัติผ่าน Web Scraping โดยแอดมินสามารถสั่งอัพเดทราคาได้ตามต้องการ ระบบจะดึงข้อมูลราคาล่าสุด วิเคราะห์และบันทึกลงฐานข้อมูลเพื่อให้ข้อมูลราคาเป็นปัจจุบันอยู่เสมอ

## Use Cases Description

### 1. ดูข้อมูลส้ม (View Orange Data)
**Actor:** ผู้ใช้งาน (User)  
**Description:** ผู้ใช้สามารถดูข้อมูลส้มแต่ละชนิด รวมถึงราคา น้ำหนัก ขนาด และคุณสมบัติต่างๆ  
**Preconditions:** ผู้ใช้เปิดแอปพลิเคชัน  
**Postconditions:** แสดงรายการส้มพร้อมข้อมูลรายละเอียด

### 2. คำนวณราคาส้ม (Calculate Orange Price)
**Actor:** ผู้ใช้งาน (User)  
**Description:** ผู้ใช้เลือกชนิดส้มและใส่น้ำหนัก ระบบจะคำนวณราคารวม  
**Preconditions:** มีข้อมูลส้มในระบบ  
**Postconditions:** แสดงราคารวมและบันทึกประวัติการคำนวณ

### 3. ดูราคาปัจจุบัน (View Live Prices)
**Actor:** ผู้ใช้งาน (User)  
**Description:** ดูราคาส้มปัจจุบันที่ดึงมาจาก Talaadthai.com  
**Preconditions:** มีการเชื่อมต่อ Backend API  
**Postconditions:** แสดงราคาล่าสุดของส้มแต่ละชนิด

### 4. ดูประวัติการคำนวณ (View Calculation History)
**Actor:** ผู้ใช้งาน (User)  
**Description:** ดูประวัติการคำนวณราคาที่ผ่านมา  
**Preconditions:** มีการคำนวณราคาอย่างน้อย 1 ครั้ง  
**Postconditions:** แสดงรายการประวัติพร้อมวันที่และราคา

### 5. ดูสถิติ (View Statistics)
**Actor:** ผู้ใช้งาน (User)  
**Description:** ดูสถิติการคำนวณ เช่น ส้มที่นิยม จำนวนการคำนวณ ฯลฯ  
**Preconditions:** มีประวัติการคำนวณ  
**Postconditions:** แสดงข้อมูลสถิติในรูปแบบกราฟและตาราง

### 6. จัดการข้อมูลส้ม (Manage Orange Data)
**Actor:** แอดมิน (Admin)  
**Description:** เพิ่ม แก้ไข หรือลบข้อมูลส้มในระบบ  
**Preconditions:** เข้าสู่ระบบด้วยสิทธิ์แอดมิน  
**Postconditions:** ข้อมูลส้มในฐานข้อมูลได้รับการอัพเดต

### 7. Scrape ราคาจากเว็บ (Scrape Prices from Web)
**Actor:** แอดมิน (Admin), External System (Talaadthai.com)  
**Description:** ดึงข้อมูลราคาส้มจาก Talaadthai.com อัตโนมัติ  
**Preconditions:** API Backend ทำงาน, เว็บ Talaadthai.com เข้าถึงได้  
**Postconditions:** ข้อมูลราคาในฐานข้อมูลได้รับการอัพเดต
